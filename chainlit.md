# Market Scout 🔍

**AI-powered Competitive Intelligence**

Track competitor product features, API updates, and releases in real time.

Built with **Google ADK · Groq LLaMA 3.3 · Tavily Search · Chainlit**

---

### Quick Examples
- `Track Stripe`
- `Tesla latest features`
- `Compare PayPal and Stripe`
- `Nike product updates`

---

### Output Files
Every run generates:
- 🌐 HTML Dashboard (persistent across runs)
- 📊 Excel workbook with charts
- 📄 PDF report
- 📝 Plain-text briefing with citations
